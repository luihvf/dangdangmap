# 🐾 댕댕여지도 (DangDangMap)

PGIS 기반 사용자 참여형 반려견 친화 공간 가이드 웹앱

## 실행 방법

```bash
npm install
npm run dev
```

## Vercel 배포

1. GitHub에 push
2. [vercel.com](https://vercel.com)에서 Import
3. 별도 설정 없이 Deploy 클릭

## 기술 스택
- Next.js 14 + TypeScript
- Tailwind CSS
- MapLibre GL + react-map-gl
- PostGIS 연동 대비 설계 (현재 Mock 데이터)
